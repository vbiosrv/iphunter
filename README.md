📖 Описание
VK Cloud Subnet Hunter — это bash-скрипт для автоматического поиска и получения публичных IP-адресов с нужными префиксами в облачной платформе VK Cloud (OpenStack).

Скрипт последовательно перебирает указанные подсети, создавая и удаляя floating IP, пока не найдет адрес, начинающийся с заданного префикса (например, российские диапазоны 89.x.x.x, 5.x.x.x и т.д.).

🚀 Быстрая установка
Шаг 1: Клонирование репозитория
bash
git clone https://github.com/idsmef/vk-cloud-subnet-hunter.git
cd vk-cloud-subnet-hunter
Шаг 2: Настройка прав
bash
chmod +x vk-subnet-hunter.sh
Шаг 3: Получение OpenRC файла
Войдите в личный кабинет VK Cloud

Перейдите: Аккаунт → API доступ

Скачайте OpenStack RC file (project-openrc.sh)

Поместите файл в директорию со скриптом:

bash
mv ~/Downloads/project-openrc.sh ./
Шаг 4: Запуск
bash
./vk-subnet-hunter.sh
⚙️ Настройка
Основные параметры (в начале скрипта)
bash
# ========== НАСТРОЙКИ ==========

# Целевые подсети (имена из openstack subnet list)
TARGET_SUBNETS=("ext-sub19" "ext-sub21" "ext-sub24" "ext-sub35" "ext-sub37" "ext-sub18")

# Искомые префиксы (в порядке приоритета)
PREFIXES=("89." "5." "95." "94." "37." "185.")

# IP, который нельзя удалять (защищенный)
SAVED_IP="90.x.x.x"

# Задержки между попытками (секунды)
MIN_DELAY=3
MAX_DELAY=7

openstack --version

env | grep OS_

# Попробуйте загрузить вручную
source project-openrc.sh

❓ Часто задаваемые вопросы
Сколько времени занимает поиск?
В среднем от 1 до 30 минут, в зависимости от:

Количества подсетей

Удачности префикса

Загруженности VK Cloud

Не заблокируют ли аккаунт?
Скрипт имитирует ручные действия с разумными задержками. Блокировка маловероятна при соблюдении настроек по умолчанию.

Сколько это стоит?
Создание floating IP — бесплатно

Использование floating IP — тарифицируется

Рекомендуется удалять неиспользуемые IP

Можно ли искать другие префиксы?
Да, просто укажите нужные в PREFIXES. Например:

bash
PREFIXES=("192.168." "10.0.")  # Для приватных сетей
Что делать с найденным IP?
bash
# Привязать к серверу
openstack floating ip set --port <PORT_ID> <IP>
openstack server add floating ip <SERVER> <IP>
